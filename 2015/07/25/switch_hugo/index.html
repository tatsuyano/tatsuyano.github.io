ブログをHugoに移行した

Posted by Tatsuyano on Sat, Jul 25, 2015
In
Tags ruby blog

今までaws上でブログ(Ghost)を運用していたが、無料期間が終わったら月2000円ちょっとかかったので、
もったいないので、またGithub Pages上にブログを移行させた。

以前はOctopressを使っていたが、最近はHugoというのが、流行っているらしいのでHugoを使ってみた。
以外と記事が多かったので、スクリプトを書いてエクスポートさせた。

以外と書くのに時間がかかったが、久々にrubyでなんか楽しかった。

#!/usr/bin/env ruby

require 'unindent'
require 'rss'
require 'sqlite3'

def make_article(db,item)
  db.execute(sprintf("SELECT id, title, markdown FROM posts WHERE slug = '%s'", item[:slug])) do |row|
    info = {
      post_id: row[0],
      title: row[1],
      markdown: row[2],
      tags: []
    }
    db.execute(sprintf("SELECT tags.name FROM posts_tags, tags WHERE posts_tags.post_id = %d AND posts_tags.tag_id = tags.id", info[:post_id])) do |row2|
      info[:tags].push row2[0]
    end

    File.open(sprintf("post/%s",item[:file]),"w") do |file|
      article = <<-EOF.unindent
        +++
        date = "#{item[:date]}"
        draft = false
        image = "/img/about-bg.jpg"
        tags = #{info[:tags]}
        title = "#{info[:title]}"
        slug = "#{item[:slug]}"
        +++
      EOF
      
      # info[:markdown]をヒアドキュメントに加えると、インデントが崩れたので、ヒアドキュメントの外で結合
      article = sprintf"%s\n%s", article, info[:markdown]
      file.puts(article)
    end
  end
end

list = []
blog = 'http://blog.10rane.com'
(1..9).each do |num|
  url = sprintf "%s/rss/%d", blog, num # e.g. http://blog.10rane.com/rss/2
  rss = RSS::Parser.parse(url)
  rss.items.each do |entry|
    all,year,month,day,slug = entry.link.match(%r|^#{blog}/(\d{4})/(\d{2})/(\d{2})/(.*)/$|).to_a

    list.push({
        slug: slug,
        year: year,
        month: month,
        day: day,
        ymd: sprintf("%s%s%s", year, month, day)
      })
  end
end

db = SQLite3::Database.new('ghost.db')
hour = 1, old_ymd = ''
list.sort!{|a,b| a[:ymd] <=> b[:ymd]}
list.each do |item|
  if old_ymd != item[:ymd] then
    hour = 1
    old_ymd = item[:ymd]
  else
    hour += 1
  end
  
  item[:file] = sprintf "%s_%s_%s_%s.md", item[:year], item[:month], item[:day], item[:slug]
  item[:date] = sprintf "%s-%s-%sT%02d:00:00+09:00", item[:year], item[:month], item[:day], hour
  make_article(db,item)
end
